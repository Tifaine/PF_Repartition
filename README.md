Programme serveur pour SMA

## Le récupérer:
```Shell
> git clone --recurse-submodules https://github.com/Tifaine/PF_Repartition
```

## Compiler:
### Première foi:
```Shell
> ./Configure/configure
> make
```

### Après:
```Sehll
> make
```

## Utilisation:
Vous pouvez lancer le programme comme suit :
```Shell
> ./bin/exec
```
